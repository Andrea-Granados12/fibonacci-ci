# Proyecto Fibonacci con CI

Este proyecto implementa la función matemática **Fibonacci(n)** utilizando Python.  
Además, incluye **pruebas automatizadas con pytest** y un **pipeline de Integración Continua (CI)** usando **GitHub Actions**.

## Función implementada

La función `fibonacci(n)` calcula el número en la posición **n** de la serie de Fibonacci.

Ejemplo:

fibonacci(10) = 55

La función también valida errores como:

- Valores negativos
- Tipos de datos incorrectos
